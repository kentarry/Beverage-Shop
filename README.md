# 🧋 飲料店開店規劃工具 (Beverage-Shop)

一個全方位的飲料店創業評估與規劃工具，幫助你從零開始評估開一間飲料店的所有成本與預期收益。

## ✨ 功能特色

- **📊 總覽儀表板** — KPI 卡片、成本圓餅圖、投資佔比、12個月現金流預測
- **🤝 合夥人設定** — 多人出資比例計算
- **👥 人事編制** — 店長/正職/工讀薪資計算（含2026年基本工資行情參考）
- **🏠 店租與裝潢** — 中和區各商圈租金行情對照表
- **⚙️ 設備清單** — 24項設備全新/二手價格比較（含購買平台建議）
- **🫧 原物料成本** — 每杯拆解成本計算
- **📍 供應商地圖** — 新北/台北地區原物料供應商，含地址與Google Maps連結
- **📋 菜單定價** — 平均售價與日銷杯數設定
- **🏪 加盟比較** — 10大品牌加盟金比較（含北投紅茶、50嵐、CoCo等）
- **💰 財務預估** — 完整月度損益表（新增週轉金、外送平台抽成、營業稅設定）
- **📈 損益兩平** — 回本月數 + 4種情境模擬
- **✅ 開店清單** — 5階段50項待辦事項

## 🚀 使用方式

### 線上使用
直接透過 GitHub Pages 開啟：https://kentarry.github.io/Beverage-Shop/

### 本地使用
```bash
# Clone 專案
git clone https://github.com/kentarry/Beverage-Shop.git

# 進入目錄
cd Beverage-Shop

# 用任何 HTTP 伺服器開啟（或直接開 index.html）
npx http-server -p 8080
```

## ⚠️ 免責聲明

- 所有金額為市場公開資訊整理，**實際金額請務必向各品牌總部/供應商直接確認**
- 加盟費用會隨時間與政策變動，本工具僅供初步評估參考
- 二手設備價格為市場行情估算，實際交易價格以雙方議定為準
- 本工具不構成任何投資建議
